🏠 Home Automation System

Abstract

Home Automation System is a smart project that allows users to control electrical appliances remotely using a mobile phone, Bluetooth module, or Wi-Fi module. It improves convenience, energy efficiency, and home security.

Components Required

  • Arduino Uno
  • Relay Module
  • Bluetooth Module (HC-05) or ESP8266
  • LED Bulb / Appliance
  • Power Supply
  • Breadboard
  • Jumper Wires

Working Principle

The user sends commands from a mobile application. The controller receives the commands and activates the relay module to switch appliances ON or OFF. This enables remote control of home devices.

Applications

  • Smart Homes
  • Remote Appliance Control
  • Energy Saving Systems
  • IoT Based Automation
