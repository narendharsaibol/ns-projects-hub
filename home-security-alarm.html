🔥 Fire and Smoke Alarm System

Abstract

The Home Security Alarm System is designed to detect unauthorized entry into homes and provide instant alerts through a buzzer. It helps improve home security and prevent theft.

Components Required

  • MQ-2 Smoke Sensor
  • Arduino Uno
  • Buzzer
  • LED
  • Breadboard
  • Jumper Wires

Working Principle

The MQ-2 sensor continuously monitors the surrounding air. When smoke is detected above a predefined threshold, the Arduino processes the signal and activates the buzzer and LED to provide an immediate warning.

Applications

  • Homes
  • Offices
  • Factories
  • Warehouses
  • Industrial Safety Systems

Future Scope

The project can be enhanced with IoT technology to send real-time alerts to mobile phones and emergency services during fire incidents.
